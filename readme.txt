== Description ==

This Wordpress widget displays a live chart of your Net Worth.
For more information, visit https://www.networthshare.com/widget

== Installation ==
1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Set your NetworthShare username (Admin > Settings > My Net Worth).
3. Enter a number (between 2 and 1440) to determine how many months to go back. The default is 6.
3. Finally, add the Widget (Appearance > Widgets > My Net Worth Widget) to your theme.
